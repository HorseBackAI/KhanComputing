Incrementing shortcut 快捷地给增量

让数变化！让参数变化！让位置（运动起来）、外观变化！

x += 1; // x = x + 1;
y -= 1; // y = y - 1;
ballWidth *= 99/100; // ballWidth = ballWidth * 99/100;
ballHeight *= 99/100; // ballHeight = ballHeight * 99/100;

Even shorter: eyeSize++;
