Text and strings 文本～字符串

// 字体颜色
fill(0, 13, 255); 

// 字号，也就是字体大小
textSize(18); 

// 输出吧文本
text("We are coding!", 99, 155);
  • text() 的参数：
    • 文本内容 content string,
    • 文本框左下角的横坐标 x of left-bottom corner of the text box
    • 文本框左下角的纵坐标 y of left-bottom corner of the text box

String 字符串 ～ 第二个数据类型！

String 字符串 is another important data type 数据类型, besides numbers 数.

用一对引号括起来！

"Enclosed with a pair of quatation marks."

Example code: 好几个张小二～～

// Set font color and size 设置字体颜色和字号
fill(92, 24, 219);
textSize(20);

// Define the string as an variable 
// 定义一个变量，它是字符串！
var myName = "张小二呀"; 

// Use the variable as the text content to display
// 使用这个变量作为文本，多次输出吧
text(myName, 40, 60);
text(myName, 40, 90);
text(myName, 40, 120);

Concatenate strings 把几个字符串：串接起来！

Example code

var myName = "张老二";
var message = myName + ": " + " 你不是张老三" + "!!";    // Concatenate 2 strings
text(message, 41, 30);
