Arrays 数组

  • Definition 定义
  • Length 长度：数组里含有多少（n）个元素
  • Indexing 索引：range from 0 to n-1

Example code: Intro to Array

// A variable
var myClassmate = "Tom";

// An array
var myClassmates = ["思齐","舒涵","一凡","洋"];

// Length of array
fill(255, 0, 0);
text("\n我有" + myClassmates.length + "名同学!!!", 10, 20);

// Indexing
text(myClassmates[0], 10, 80);
text(myClassmates[1], 10, 110);
text(myClassmates[2], 10, 140);
text(myClassmates[3], 10, 170);
text(myClassmates[4], 10, 200); // Error: Indexing is out of the range

Looping through array 循环（爱）数组（的元素）

Loop through each element of the array. Increment the index at each loop.

Example code: Looping through arrays

var myClassmates = ["思齐","舒涵","一凡","洋"];

fill(255, 0, 0);

var i = 0;
while(i < myClassmates.length) {
    text(myClassmates[i], 50, 60+i*30);
    i++;
}

for(var i = 0; i < myClassmates.length; i++) {
    text(myClassmates[i], 150, 60+i*30);
}

Modify element in array 修改数组里元素的值

array[index of element] = updated value

myClassmates[3] = "澎";

Push new element into array 添加新元素到数组

array.push(new element)

myClassmates.push("洋");

New element is added at the end of array. 新元素添加为数组的最后的一个元素。

Example code: Modifying Arrays - Modify element; Push new element into array

// Modify one element
xPositions[1] = 300;
    
// Push new element into array
if(mouseIsPressed){
    xPositions.push(mouseX);
}

Project: Make it rain

Example code: make it rain

// Add more drops if mouse clicked
mouseClicked = function() {
  xPositions.push(mouseX);
  yPositions.push(mouseY);
  dropColors.push(color(random(0, 255), random(0, 255), random(0, 255)));
};

draw = function() {
    background(204, 247, 255);

    noStroke();

    for (var i = 0; i < xPositions.length; i++) {
        fill(dropColors[i]);
        ellipse(xPositions[i], yPositions[i], 10, 10);
        if (yPositions[i] >= 400) {
            yPositions[i] = 0;
        }
        yPositions[i] += 3;
    }
};
