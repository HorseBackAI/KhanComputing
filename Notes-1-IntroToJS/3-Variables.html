Variables 变～～量

Specify the eye size, making them shining.

1) Define a variable

Declare and then assign it a value.

var eyeSize; 
eyeSize = 30;

or

var eyeSize = 30;

  • Think var as a bucket.
  • = is not an equal sign, but assignment.
  • Assign a value to the variable. Put a value into the bucket.

2) Use it

ellipse(157, 151, eyeSize, eyeSize);
ellipse(304, 142, eyeSize, eyeSize);

So: 1) define, 2) use a variable to make the eyes same size or shiny.

Order matters!

Define first, and then use it.

Slider is cool!

Variables make things move easy!

  • Elements of the face -- Find the base point or line. Make the base point or line as a variable.
  • Express element positions relative to the base point or line.
  • One (the variable) changes, others follow! Move the face so easy ~~
var eyeSize = 40;
var x = 150;

// face
...
ellipse(x, 208, 300, 300);

// eyes
...
ellipse(x - 50, 151, eyeSize, eyeSize);
ellipse(x + 100, 142, eyeSize, eyeSize);

// mouth
...
ellipse(x + 50, 240, 120, 136);

Variable names

  • Begin with letters a-z, or $ or _; Cannot begin with a number 0-9.
  • Case sensitive 区分大小写的！xPos and xpos are 2 variables.
  • Clear and meaningful!
  • Camel case for multiple words, like toothSize instead of tothsize or tooth_size.
