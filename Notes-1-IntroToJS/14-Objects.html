如果有好多个小明～～同学

同学们的共有信息： Student:

  • name
  • age
  • shoeSize
  • ...

那么，可以把同学自定义为一种数据类型，包括了上述的信息 这种自定义的数据类型：

  • 更符合直觉
  • 更方便使用

Example code: 对象的类型和实例

/* 
Type 数据类型: 
Student 首字母大写即可
constructor function 构造数据类型的方法
就是定义一个function
每个同学的信息呀有不同的值，它们就是参数，输入给function
那么输出的就是不同的每位具体的同学
*/
var Student = function(name, age, shoeSize, x, y) {
    this.name = name;
    this.age = age + "岁";
    this.shoeSize = "鞋码" + shoeSize + "号";
    this.x = x;
    this.y = y;
};

/*
Instance 实例：
使用同学这个自定义的数据类型的构造方法，
给出具体参数，构造出各不相同的一个个新的对象（实例）
*/
var xiaoming = new Student("小明", 13, 43, 20, 150);
var azhen = new Student("阿珍", 12, 40, 229, 150);

fill(255, 0, 0);
text(xiaoming, 30, 30);
text(xiaoming.name + ': ' + xiaoming.shoeSize, 30, 60);

Modifying Objects 修改更新对象

Example code: Modify objects

定义对象类型

// Define object Type 定义对象类型
var Student = function(name, age, sports, studyProgramming, contact) {
    this.name = name;
    this.age = age;
    this.sports = sports;
    this.studyProgramming = studyProgramming;
    this.contact = contact;
};

// Create a new object instance 创建新的对象实例

Array of Objects 一组对象推进一个数组

回归小结：数组和对象的用途

  • Array 数组：方便循环，自动执行
  • Object 对象：方便整合信息，每个事物、也就是每个对象其实都是一个信息的集合

How to draw these faces? 怎样画这么老一些脸？

  • Method 1: One by one, calling the draw function and giving x and y values. 一个一个画，每一次手动调用画画方法
  • Method 2: Loop x and y arrays 把坐标弄成数组，自动循环画
  • Method 3: Make positions as objects, and change draw method to accept position object as input把位置做成对象，改造画画方法接收对象作为输入的参数
  • Method 4: Push position objects into array and loop through the array 推一组位置对象进数组，自动循环画

Example code: Array of Objects

// Push a few face position objects into a array
// 把一组位置对象push进一个数组
var facePositionArray = [
    {x: 99, y: 105},
    {x: 294, y: 105},
    {x: 101, y: 302},
    {x: 294, y: 302},
    {x: 200, y: 200}
];

// Change drawWinston method to accept a position object as input
// 改造drawWinston方法，让它接收的参数是一个位置对象
var drawWinstonObj = function(facePosition) {
    var faceX = facePosition.x;
    var faceY = facePosition.y;
    
    fill(255, 255, 0);
    ellipse(faceX, faceY, 180, 180); // face
    fill(46, 46, 41);
    ellipse(faceX - 30, faceY - 50, 30, 30); // eyes
    ellipse(faceX + 44, faceY - 55, 30, 30);
    fill(252, 65, 65);
    ellipse(faceX + 19, faceY + 32, 82, 81); // mouth
};

for (var i = 0; i < facePositionArray.length; i++) {
    drawWinstonObj(facePositionArray[i]);
}
